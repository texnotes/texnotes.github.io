Реализация плагина табов tabable.

Отлично, что на домашку была целая неделя. Понемногу думал, как буду реализовывать,
но большой завал на работе привел к тому, что за домашку сел только в пятницу.
По-этому решил использовать стили Bootstrap для ускорения работы и реализовал 
плагин только для первого элемента коллекции. Очевидно, для всей коллекции 
прийдется хранить ссылки на каждый созданный класс. 

Плагин не манипулирует инлайновыми стилями (только классами). 

Не реализовал событие Hide, т.к. интересно насколько на текущий момент понял 
концепцию общей логики построения плагина и идею использования кастомных событий. 
Собственно, реализовать Hide - технически просто т.к. есть предыдущее состояние 
активного таба.
